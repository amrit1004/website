# Claude Code Guidelines

## Git Commits

When making git commits, always use the following identity for both author and committer:

```
git commit --author="salvoshootersarenamohali-creator <salvoshootersarenamohali@gmail.com>"
```

Also set the committer environment variables:

```
GIT_COMMITTER_NAME="salvoshootersarenamohali-creator" GIT_COMMITTER_EMAIL="salvoshootersarenamohali@gmail.com" git commit --author="salvoshootersarenamohali-creator <salvoshootersarenamohali@gmail.com>"
```
